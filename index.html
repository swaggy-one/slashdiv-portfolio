About me

I'm a server developer and manager. I have worked on countless online projects since I got into server development in 2021.

My role in Minecraft development is to create a stable and fun server. I can optimize configs and research plugins to make sure your server runs well. I have created three servers and entirely managed another successful server for a Twitch streamer (ZonoSMP).

One of the networks I am currently working on, PvPHub, has over 6,000 Discord members. I am supporting the developer of one of their upcoming modes as they learn the basics of server development.

I have expereince in Linux. If you for example, get a dedicated server, I can set it up! I have learned this from setting up Oracle Cloud accounts for many small SMPs to save them money and resources.
